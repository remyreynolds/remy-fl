changes every 1–2 bars.

## Chord formulas (scale degrees)
- Major triad: 1–3–5
- Minor triad: 1–♭3–5
- Dim: 1–♭3–♭5
- Sus2: 1–2–5 · Sus4: 1–4–5
- Aug: 1–3–♯5
- Maj7: 1–3–5–7 · Min7: 1–♭3–5–♭7 · Dom7: 1–3–5–♭7
- Half-dim7: 1–♭3–♭5–♭7 · Dim7: 1–♭3–♭5–♭♭7

## Diatonic qualities
**Major key (1–7):** Maj, min, min, Maj, Maj, min, dim  
**Minor key (1–7):** min, dim, Maj, min, min, Maj, Maj  

**Major 7ths:** Maj7, min7, min7, Maj7, Dom7, min7, half-dim7  
**Minor 7ths:** min7, half-dim7, Maj7, min7, min7, Maj7, Dom7

## House production tips for MIDI
1. Nail chords before melody/bass color.
2. Split progression notes across roles (chords mid, bass roots, melody top contour).
3. Learn the loop, then break one rule deliberately (one borrowed chord or one syncopation).
4. Analyze reference MIDI — recreate pocket, don’t clone melodies.
