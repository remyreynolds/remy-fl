# Electronic music workflow — MIDI slice

Creation → arrangement → (sound design/mix later). This agent owns **creation
of MIDI roles**.

## Three actions of production (MIDI view)
1. **Create** — write loopable ideas: chords, bass, melody/arp, perc
2. **Arrange** — decide density across bars; leave breath bars
3. **Define roles** — each track has one job; don’t double the bass root in
   dense chord blocks unless intentional

## Practical loop recipe (8 bars)
1. Lock key + BPM + swing from session / genre card
2. Place chord loop (1–4 chords; change every 1–2 bars)
3. Bass follows roots ≥70%; pick archetype (offbeat / rolling / sub_hold /
   walking)
4. Melody or arp uses chord tones + motif repetition
5. Perc/hat grid supports swing; don’t overfill

## Mindset for the agent
- Finish a playable loop; don’t stall on theory dumps
- Prefer authentic house pocket over novel complexity
- One clear signature move beats five competing ideas
- When revising: change one layer at a time
