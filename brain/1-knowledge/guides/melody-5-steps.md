# Writing melodies — 5 steps (MIDI agent)

Before generating a melody or arp MIDI loop:

1. **Choose a scale** — limitations create identity. Prefer natural minor,
   Dorian, or minor pentatonic for house; match session key.
2. **Write the rhythm first** — horizontal placement before pitches. Leave
   space on bar 4/8. Prefer 2–4 note cells that repeat.
3. **Draw a contour** — decide rise / fall / arch before filling notes.
   Keep adjacent top notes within a 3rd when over chords.
4. **Choose the sound intent** — pluck/lead/keys changes gate length and
   register; obey active sound-type constraints.
5. **Write** — place pitches on the rhythm using chord tones on strong beats
   and passing tones on weak beats. Motif first, variation every 4–8 bars.

Memorable = simple rhythm + clear contour + repetition with micro-variation.
